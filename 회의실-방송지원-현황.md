# 회의실 방송지원 현황

이 문서는 각 회의실별 방송장비 지원 현황을 정리한 자료입니다.

---

## 중회의실 방송지원 현황

### 방송지원

- **유선 마이크**: 4개 (사회석 2개, 의장석 2개)
- **무선 마이크**: 4개
- **이어 마이크**: 1개 (이어 마이크 사용시 무선핸드는 3개만 사용가능)
- **회의용 마이크**: 25개 (목책상만 설치 가능)
- **LED 전광판**: 사용자 노트북, HDMI 연결
- **녹음, 녹화**: USB 4G 이상 지참, 1층 방송운영단에서 수령
- **국민의례, 농협의 노래**: 음원
- **식전, 시상식 음악**: 시나리오에 별도 표기 요청
- **DID 전자 식순**: 총무팀 (6010)
- **냉·난방 지원**: 관제실 (5119)

### 시설 현황

- **LED 전광판 사이즈**: 160인치 (가로 3.6m x 세로 2m)
- **현수막 사이즈**: 가로 6m x 세로 0.7m
- **무대 사이즈**: 가로 13m x 세로 3m
- **최대 수용 인원**: 200명
- **개별 냉·난방**: 가능
- **인터넷**: 내부망 / 외부망

## 화상회의실 방송지원 현황

### 방송지원

- **유선 마이크**: 1개 (사회석)
- **무선마이크**: 2개
- **회의용마이크**: 31개 (고정형, 이동 불가)
- **녹음, 녹화**: 4G 이상 USB 지참, 1층 방송실에서 수령 가능
- **LED 전광판**: 사용자 노트북 HDMI 연결
- **국민의례, 식전음악**: 지원
- **전자명패**: 지원
- **전자교탁**: 지원
- **신 화상회의 시스템**: zoom회의 불가

> **참고**: 전자 명패, 전자교탁, 화상회의 사용시 총무팀(6010)으로 사전 문의 바랍니다.

### 시설현황

- **LED 전광판 사이즈**: 160인치 (가로 3.6m x 세로 2m)
- **현수막 사이즈**: 560cm x 55cm
- **최대 수용 인원**: 61명 (고정)
- **개별 냉·난방**: 가능
- **인터넷**: 내부망 / 외부망

### 주의사항

- 화상회의실 신 화상회의 시스템은 회의실 사용자 측이 직접 운영합니다.
- 화상회의 진행 중 시스템 내 장애 발생, 참석자 접속 설정 방법 미숙으로 인한 소리울림 또는 잡음이 일어나지 않도록 미리 테스트하셔야 합니다.
- 신 화상회의 시스템 사용 시 최소 3일전 사전 리허설 시간을 예약하여 충분히 확인 후 진행 하셔야 합니다.
- ZOOM 화상회의는 현재 지원이 되지 않습니다.
- 화상회의실 회의용 장비는 모두 고정형으로써 이동 배치가 불가합니다.

> **참고**: 의장석 이동, 자리 배치 변경을 원하실 경우 중회의실을 사용하셔야 합니다.

## 소회의실 방송지원 현황

### 방송지원

- **유선 마이크**: 2개 (사회석 1개, 의장석 1개)
- **무선마이크**: 2개
- **전자칠판**: 75인치 / 노트북 HDMI 연결 / 사운드 자체 출력
- **국민의례, 녹음, 녹화**: 불가
- **회의용 마이크**: 없음 (마이크가 다수 필요한 경우 중회의실, 화상회의실 사용바람)

### 시설현황

- **현수막 사이즈**: 480cm x 55cm (자석 부착)
- **냉·난방**: 중앙관제실 (5119)

## 경영전략회의실 방송지원 현황

### 방송지원

- **유선 마이크**: 1개 (사회석 1개)
- **회의용 마이크**: 21개 (고정형으로 이동불가)
- **무선마이크**: 2개
- **국민의례, 녹음, 녹화**: 불가

> **참고**: 회의장 사용시 기획실과 별도 협의 (기획팀: 5129)
