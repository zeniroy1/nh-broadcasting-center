## 📄 주요 파일

### 문서 파일

- **회의실-방송지원-현황.md** - 각 회의실별 방송장비 지원 현황 통합 문서
- **list.txt** - 방송운영단 기본 이용 안내 (블로그 자료)

### 설정 파일

- **package.json** - Node.js 프로젝트 설정
- **vite.config.js** - Vite 빌드 설정
- **eslint.config.js** - ESLint 설정

### 스크립트

- **process_images.py** - 이미지 처리 스크립트
