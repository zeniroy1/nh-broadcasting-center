# 연차 계산 Excel 수식 완전 가이드 (수정본)

## 📌 핵심 원칙
- **근속기간 (E열)**: TODAY() 기준 → 실제 근무 기간 표시
- **근속연수 (F열)**: $C$2 (12/31) 기준 → 연차 계산용

---

## 🔧 최종 수식

### E열: 근속기간 (TODAY 기준)
```
=IF(OR(D5="",NOT(ISNUMBER(D5))),"",IF(DATEDIF(D5,TODAY(),"Y")>=1,DATEDIF(D5,TODAY(),"Y")&"년 "&DATEDIF(D5,TODAY(),"YM")&"개월 "&DATEDIF(D5,TODAY(),"MD")&"일",IF(DATEDIF(D5,TODAY(),"M")>=1,DATEDIF(D5,TODAY(),"M")&"개월 "&DATEDIF(D5,TODAY(),"MD")&"일",DATEDIF(D5,TODAY(),"D")&"일")))
```

### F열: 근속연수 (12/31 기준)
```
=IF(OR(D5="",NOT(ISNUMBER(D5))),"",DATEDIF(D5,$C$2,"Y"))
```

### G열: 기본 연차
```
=IF(OR(D5="",NOT(ISNUMBER(D5))),"",IF(F5=0,0,15))
```

### H열: 가산 연차
```
=IF(OR(D5="",NOT(ISNUMBER(D5))),"",IF(F5=0,MIN(11,DATEDIF(D5,$C$2+1,"M")),IF(F5>=2,MIN(10,INT((F5-1)/2)),0)))
```

### I열: 합계
```
=IF(OR(D5="",NOT(ISNUMBER(D5))),"",IF(F5=0,H5,MIN(25,G5+H5)))
```

---

## 📝 적용 방법
1. E5에 근속기간 수식 입력 → E8, E11... 복사
2. F5에 근속연수 수식 입력 → F8, F11... 복사
3. G, H, I열은 기존 수식 유지 또는 위 수식 적용
